# Exercícios de C# - Jornada de Aprendizado

Este repositório foi criado para armazenar meus exercícios, desafios e projetos práticos desenvolvidos durante meus estudos da linguagem **C#** e da plataforma **.NET**.

## Sobre o Repositório

O objetivo aqui é documentar minha evolução na lógica de programação e na sintaxe do C#. Os exercícios abrangem desde conceitos básicos até estruturas mais complexas.

### Tecnologias Utilizadas
* **Linguagem:** C#
* **Ambiente:** Visual Studio Code / .NET SDK 8.0+
* **Versionamento:** Git & GitHub
